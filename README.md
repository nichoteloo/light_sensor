# Light_Sensor
Basic project. Testing device. Light sensor that will automatically turn on and off in dark environment.

Component List :

Arduino Uno R3 (1)
Red LED (1)
Photoresistor (1)
220 Ω Resistor (1)
4.7 kΩ Resistor (1)
